# Product APP


## Dev

1. Instalar dependencias `npm install` 
2. Clonar el .env.template a .env y cambiar la direccion ip
3. Ejecutar `npm start` ó `npx expo start --tunnel -c`
4. leer la documentación de secure store y para entende como se almacena los tokens `https://docs.expo.dev/versions/latest/sdk/securestore/`
